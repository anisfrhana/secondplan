# Merchandise Product Images

Place product images in this directory with the following specifications:

## Image Requirements

- **Format:** JPG or PNG
- **Dimensions:** 800x800px (square, 1:1 ratio)
- **Background:** White or transparent
- **Quality:** Web-optimized (< 200KB per image)

## File Naming Convention

Use the product SKU as the filename:

| SKU | Filename |
|-----|----------|
| SP-TEE-BLK | `SP-TEE-BLK.jpg` |
| SP-HOOD-NVY | `SP-HOOD-NVY.jpg` |
